To print

To configure this module you must go to Settings -> General Settings -> and Select Thermal Printers.

If then we go to accounting and select one of your invoices to print. When opening it you must select from the central menu print-> invoice ticket, as shown in the image.

You will be able to run real time statistics and consolidations across all your shops without the hassle of integrating several applications.

Print with different printers

To add another printer, go to Settings -> General Settings -> and Select Thermal Printers register another ip address in Printer Urls.

Then, we go to accounting and select one of your invoices to see the options of available printers, for this we access the central menu in the option Thermal printers as it appears in the following image.

Finally repeat the previous step.
