MercadoLibre Uploader - melisync

MercadoLibre Uploader / Publisher for Odoo.

Compatible with the following versions: Odoo 12.0 Odoo 13.0 Odoo 14.0 Odoo 15.0

What does this module do?

Melisync is a module that allows you to publish (and keep synchronized) your Odoo products in the MercadoLibre Marketplace. Also, it will allow you to have several instances, in the event that your company has several branches in different countries, and thus be able to have your products published in all the corresponding countries.

There is a specific model for MercadoLibre categories (since they depend on the site id to which each instance belongs), and they have their own attributes, which can be configured in the product file.

Automatic download of the sales made in MercadoLibre, with the customer's data to be able to issue an invoice from Odoo.

The following actions are automatically scheduled to run:

  • All instances publish ready products, it will run at 00:00 hours.
  • All instances sync products stock, it will run at 01:00 hours.
  • All instances sync products all, it will run at 02:00 hours.
  • All instances sync sales, it will run at 03:00 hours.

Permission groups were created with the necessary permissions to be able to moderate, manage products and/or manage the configurations of MercadoLibre instances.
