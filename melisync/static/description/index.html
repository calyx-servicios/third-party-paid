MercadoLibre Uploader / Publisher for Odoo - melisync

Powered by Calyx Servicios S.A

This is a module that allows you to publish (and keep synchronized) your Odoo products in the MercadoLibre Marketplace. Also, it will allow you to have several instances, in the event that your company has several branches in different countries, and thus be able to have your products published in all the corresponding countries.

Features

Multiple instances of MercadoLibre

Automatic token/refresh token obtaining through authentication. Automatic obtaining of publication types based on the selected country. Currency specification. Test mode for trial posts. Stock settings - Choose warehouse to obtain product stock.

Synchronize sales orders and invoices.

Automatic download of the sales made in MercadoLibre, with the customer's data to be able to issue an invoice from Odoo.

Publishing products from Odoo to Mercado Libre

You can publish all your products with complete settings, keep your stock and prices updated, and ensure that customers always find accurate information about your products.

Button to update all imported products according to their status.

Single or multiple publication of Odoo products.

Odoo Community and Enterprise

Works with both Community and Enterprise editions (self-hosted or odoo.sh).

The system updates the publication status, stock, price and attributes.

The following actions are automatically scheduled to run: publish ready products, sync product stock, sync all product data, and sync sales data.

Products closed in MercadoLibre are no longer published in the Odoo store.

Possibility to link Product to several Publications

Possibility of linking Products type Kits to Publications

Size guide for Retail incorporated

Detailed features

MercadoLibre instances

  • Multiple instances.
  • Automatic token/refresh token obtaining just by authenticating.
  • Automatic obtaining of publication types depending on the country selected in the instance.
  • Currency specification.
  • Test mode: posts with name allowed for tests.
  • Stock settings: Choose warehouse to obtain the stock of the product

Price Settings

  • Warehouse to obtain the stock of the product.
  • Price list (or create a new one from the configuration, with a default name).
  • The number of days back from which MercadoLibre sales will be synchronized.
  • Auto synchronization: enable auto sync to update products prices, stocks, publications, unpublications of products or disable and toggle manually.
  • Create test users with one click.
  • Categories synchronization:
    • Synchronize all categories synchronously.
    • Synchronize all categories asynchronously, using configurable multiprocess
  • Shipping methods and modes: Select shipping modes, methods and auto free methods

MercadoLibre instance actions

  • Publish all products (with mercadolibre settings complete).
  • Update published products (stock / price).
  • Update published products stock.
  • Update published products price.
  • Download categories.
  • Download sales.
  • Create test users.
  • View tree of:
    • Sale orders.
    • Categories.
    • Published products.

Products settings

You can publish products with their image, price, description, attributes, etc.
In each product tab, you can configure (among other things) the following: MercadoLibre instance, category, listing type, buying info, youtube url, description, product attr by MercadoLibre category.

MercadoLibre category settings

There is a specific model for MercadoLibre categories (since they depend on the site id to which each instance belongs), and they have their own attributes, which can be configured in the product file.

User permissions

Permission groups were created with the necessary permissions to be able to moderate, manage products and/or manage the configurations of MercadoLibre instances.

Sales download

Automatic download of the sales made in MercadoLibre, with the customer's data to be able to issue an invoice from Odoo.

Schedule actions

The following actions are automatically scheduled to run:

  • All instances publish ready products, it will run at 00:00 hours.
  • All instances sync products stock, it will run at 01:00 hours.
  • All instances sync products all, it will run at 02:00 hours.
  • All instances sync sales, it will run at 03:00 hours.

Compatibility

  • Odoo Community and Enterprise
  • All Calyx custom modules products, and more
  • 3rd party Odoo apps developed according to Odoo standards
