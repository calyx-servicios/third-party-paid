MercadoLibre Uploader / Publisher for Odoo - melisync

Powered by Calyx Servicios S.A

Este es un módulo que te permite publicar (y mantener sincronizados) tus productos de Odoo en el MercadoLibre Marketplace. Además, te permitirá tener varias instancias, en caso de que tu empresa tenga varias sucursales en diferentes países, y de esta manera poder tener tus productos publicados en todos los países correspondientes.

Features

Múltiples instancias de MercadoLibre

Obtención automática de token/refresh token a través de la autenticación. Obtención automática de tipos de publicación basados en el país seleccionado. Especificación de moneda. Modo de prueba para publicaciones de prueba. Configuraciones de inventario: elige el almacén para obtener el stock del producto.

Sincronizar órdenes de venta y facturas.

Descarga automática de las ventas realizadas en MercadoLibre, con los datos del cliente para poder emitir una factura desde Odoo.

Publicación de productos desde Odoo a Mercado Libre.

Puedes publicar todos tus productos con configuraciones completas, mantener tu stock y precios actualizados, y asegurarte de que los clientes siempre encuentren información precisa sobre tus productos.

Botón para actualizar todos los productos importados según su estado.

Publicación individual o múltiple de productos de Odoo.

Odoo Community y Enterprise

Funciona tanto con las versiones Community como Enterprise (autoalojadas o odoo.sh).

El sistema actualiza el estado de publicación, el stock, el precio y los atributos.

Las siguientes acciones se programan automáticamente para ejecutarse: publicar productos listos, sincronizar stock de productos, sincronizar todos los datos de productos y sincronizar datos de ventas.

Los productos cerrados en MercadoLibre ya no se publican en la tienda de Odoo.

Posibilidad de vincular un producto a varias publicaciones.

Posibilidad de vincular productos tipo Kits a publicaciones.

Guía de tallas para ventas minoristas incorporada.

Características detalladas

MercadoLibre instances

  • Múltiples instancias.
  • Obtención automática de token/refresh token simplemente al autenticar.
  • Obtención automática de tipos de publicación según el país seleccionado en la instancia.
  • Especificación de moneda.
  • Modo de prueba: publicaciones con nombre permitido para pruebas.
  • Configuración de stock: Elige el almacén para obtener el stock del producto.

Ajustes de precios

  • Almacén para obtener el stock del producto.
  • Lista de precios (o crear una nueva desde la configuración, con un nombre predeterminado).
  • Número de días hacia atrás desde los cuales se sincronizarán las ventas de MercadoLibre.
  • Sincronización automática: habilita la sincronización automática para actualizar precios de productos, stocks, publicaciones, despublicaciones de productos, o deshabilita y activa manualmente.
  • Crear usuarios de prueba con un solo clic.
  • Sincronización de categorías:
    • Sincronizar todas las categorías de manera síncrona.
    • Sincronizar todas las categorías de manera asíncrona, utilizando multiproceso configurable.
  • Métodos y modos de envío: Seleccionar modos de envío, métodos y métodos de envío gratuitos automáticos.

Acciones de instancia de MercadoLibre

  • Publicar todos los productos (con configuración completa de MercadoLibre).
  • Actualizar productos publicados (stock / precio).
  • Actualizar el stock de los productos publicados.
  • Actualizar el precio de los productos publicados.
  • Descargar categorías.
  • Descargar ventas.
  • Crear usuarios de prueba.
  • Ver árbol de:
    • Órdenes de venta.
    • Categorías.
    • Productos publicados.

Ajustes de productos

Puedes publicar productos con su imagen, precio, descripción, atributos, etc.
En cada pestaña de producto, puedes configurar (entre otras cosas) lo siguiente: instancia de MercadoLibre, categoría, tipo de listado, información de compra, URL de YouTube, descripción, atributos de producto según la categoría de MercadoLibre.

Ajustes de categoría de MercadoLibre

Existe un modelo específico para las categorías de MercadoLibre (ya que dependen del ID del sitio al que pertenece cada instancia), y tienen sus propios atributos, que se pueden configurar en el archivo del producto.

Permisos de usuario

Se crearon grupos de permisos con los permisos necesarios para poder moderar, gestionar productos y/o gestionar las configuraciones de las instancias de MercadoLibre.

Descarga de ventas

Descarga automática de las ventas realizadas en MercadoLibre, con los datos del cliente para poder emitir una factura desde Odoo.

Acciones automatizadas

Las siguientes acciones están programadas automáticamente para ejecutarse:

  • Todas las instancias publicarán productos listos, se ejecutará a las 00:00 horas.
  • Todas las instancias sincronizarán el stock de productos, se ejecutará a las 01:00 horas.
  • Todas las instancias sincronizarán todos los productos, se ejecutará a las 02:00 horas.
  • Todas las instancias sincronizarán ventas, se ejecutará a las 03:00 horas.

Compatibilidad

  • Todos los productos de los módulos personalizados de Calyx, y más
  • Aplicaciones de terceros para Odoo desarrolladas según los estándares de Odoo.
